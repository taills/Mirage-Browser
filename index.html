    幻境浏览器 · Mirage Browser

    版本 dev version 检查更新中… 🆕 下载新版

    多开浏览器环境管理工具，支持指纹隔离、代理配置与脚本注入。

    项目说明

    本工具用于管理多个相互隔离的 Chrome 浏览器实例，每个实例独立维护：

    • 浏览器指纹（User-Agent、Canvas、WebGL、语言、时区等）
    • 代理网络（支持 HTTP/SOCKS5 直连与 Mihomo 订阅）
    • 独立用户数据目录，Cookie / localStorage 完全隔离
    • 自动化脚本注入（CDP 协议）

    指纹检测参考：

    • https://scrapfly.io/web-scraping-tools/gpu-fingerprint
    • https://github.com/mumuy/browser/
